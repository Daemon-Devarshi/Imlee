Imlee Help Guide

General:

Imlee is a simple application which is used to change the type of an image into any of these:

  1. png
  2. jpeg
  3. jpg
  4. bmp
  5. tiff
  6. gif

Action Items:

To change an image's type you need to perform following actions:

  1. drag and drop an image on image view
  2. choose any of the desired type from pop up
  3. click on save button

You can open an image by clicking on image view.

Others:

Imlee is backward compatible to Mac OS X 10.6, is tested and works fine on 10.6 and 10.7.
